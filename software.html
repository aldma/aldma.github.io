Software

A list of prototype software packages developed for performing numerical experiments on various research projects.
  • Convex quadratic programming:
    • QPDO: the Quadratic Primal-Dual Optimizer
      [ code ]
      A research C-code for solving convex quadratic programs subject to linear constraints. The method relies on a primal-dual proximal regularization of the original problem and a tailored damped semismooth Newton's method.
      For more details, refer to On a Primal-Dual Newton Proximal ...

  • Equality-constrained nonlinear programming:
    • HALeqO: Homotopy Augmented Lagrangian method for EQuality-constrained Optimization
      [ code ]
      A research Julia-code for solving continuous nonlinear equality-constrained optimization problems, based on a homotopy augmented Lagrangian method and globalised Newton's steps with Armijo's line-search.
      For more details, refer to Augmented Lagrangian methods as ...

  • Constrained structured programming:
    • Bazinga: a toolbox for constrained structured optimization
      [ code ]
      A research Julia-code for solving nonsmooth, nonconvex, constrained, structured optimization problems. This problem class is investigated in my doctoral dissertation, as well as the algorithmic framework based on augmented Lagrangian and proximal methods.
