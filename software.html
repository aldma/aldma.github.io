Software

A list of prototype software packages developed for performing numerical experiments on various research projects. For more details, other projects and personal code please visit my GitHub account.
  • QPDO
    The Quadratic Primal-Dual Optimizer (QPDO) solver is a numerical optimization package for convex quadratic programming.
    [ code | paper ]
    Written in C, with MATLAB interface.
    The method relies on a primal-dual proximal regularization of the original problem and a tailored damped semismooth Newton's method.

  • HALeqO.jl
    A numerical optimization package implementing a Homotopy Augmented Lagrangian method for targeting EQuality-constrained nonlinear Optimization problems.
    [ code | paper ]
    Written in Julia. The method relies on primal-dual proximal regularization and globalised Newton's steps with Armijo's line-search.

  • Bazinga.jl
    A toolbox for constrained composite programming: nonsmooth, nonconvex, nonlinear, constrained optimization problems.
    [ code | phdthesis | paper ]
    Written in Julia. The class of constrained composite optimization problem was investigated in my doctoral dissertation and in a subsequent article, as well as the algorithmic framework based on augmented Lagrangian and proximal methods.
